# Задача task#fe5d

```

    LOCK();
    A = READ();
    A = A + 1;
    WRITE(A);
    UNLOCK();
}
```

Необходимо написать примитивную реализацию этой абстракции на PHP. Какие допущения при этому будут сделаны?
